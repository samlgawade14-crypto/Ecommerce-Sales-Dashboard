# E-Commerce Sales Dashboard 2025

This project analyzes e-commerce sales data using Python.

Project Overview :

This project analyzes an Indian E-Commerce sales dataset and creates an interactive sales dashboard using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

The project performs data cleaning, exploratory data analysis (EDA), and visualization to generate business insights from the sales data.

Objective :

Clean and preprocess the sales dataset.
Analyze sales and profit trends.
Visualize important business metrics.
Generate useful insights for decision making.
